Genesis Menu options with custom menu, categories, pages, and primary/secondary nav extras.

== Description ==

The Genesis Simple Plugin takes all the great Genesis plugins available and consolidates them into a single plugin.  This allows for code simplicity and efficiency.  There is a "Jet Pack" style interface to allow individual modules to be enabled or disabled as needed.  


This plugin requires the [Genesis Theme Framework](http://designsbynickthegeek.com/go/genesis) aff link
